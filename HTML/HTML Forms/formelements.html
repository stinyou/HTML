We have already covered the <input> and <label> elements.
Next, the <select> element is used to display a drop-down list.

By default, the first value in the drop-down list is the one automatically selected.
To change this, add the selected attribute to the one you would like to be the automatic.
Can use the size attribute to specify how many visible values to be shown.
Can use the multiple attribute so that the user can select multiple values.
The user must use Ctrl + click to select multiple values.
Can use the <textarea> element to provide a larger input field.
The <button> element is used to provide a clickable button.

Can use <fieldset> to group related elements and <legend> to provide a caption.
