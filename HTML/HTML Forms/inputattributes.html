We have already seen some attributes for the input element, like type, id, name, and value. The readonly attribute means that the input cannot be modified by the user. The input is still sent when the form is submitted.






The disabled attribute makes an input field unmodifiable and unclickable. The input is not sent when the form is submitted.





The size attribute defines the width of the input field, in characters. It only works on certain input types.





The maxlength attribute defines a maximum number of characters the user can input. The user is not notified of this without the help of JavaScript.





The min and max attributes set a minimum and maximum value for the input field. This only works with certain input types.

The pattern attribute specifies a form the input must follow, checked against when submitted.

The placeholder attribute specifies a hint at what form the input should follow.

The required attribute means the field must be filled out before submitting a form.





The step attribute defines intervals of acceptable values for a field.

The autofocus attribute defines a field that should automatically get focused on when loading the page.
