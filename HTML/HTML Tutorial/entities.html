Some characters in HTML are reserved and must be replaced with entities. For example, must use < for less than and > for greater than symbols. This is especially useful when writing HTML elements like <div>.

There are entity names and entity numbers in HTML. For less than, < is the entity name and < is the entity number.

Another useful entity is the non-breaking space. This allows text to remain together even if there is a new line that would separate them. For example, may want to use it on something like 20 km/h. Can also, use non-breaking spaces to write multiple spaces in a row.

Similarly, can use ‑ for a non-breaking hyphen.

Diacritical marks are "glyphs" added to letters, such as accents. Can combine these with alphanumeric characters to create characters that are not present it the usual set. For example, can write é and Õ.
