The <section> element is used to define a section of content. It usually will have a header.

For example, sections can be used for:

  • Table of Contents
  • Introduction
  • Main Content
  • Contact Information

The <article> element is used for content that makes sense on its own.

For example, articles can be used for:

  • User Comments
  • Articles
  • Social Media Posts

The <header> element is used for introducing the webpage or providing links.

The header can contain:

  • Heading Elements
  • Logo/Icon
  • Authors

The <footer> element is used for footers of a document/section.

The footer can contain:

  • Authors
  • Contact Information
  • Copyright Information
  • Back to Top Navigation
  • Related Documents

The <nav> element is used for a set of navigation links.

The <aside> element is used for content that is somewhat related to the content around it, like a sidebar.
