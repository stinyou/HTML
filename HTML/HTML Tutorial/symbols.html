Symbols can be added using entities as well.
Some examples are:

  • For all: ∀
  • There exists: ∃
  • Sum: ∑
  • Alpha: Α
  • alpha: α
  • Pi: Π
  • pi: π
  • Copyright: ©
  • Diamond: ♦
  • Euro: €

All of these symbols were written using their entity name, but could use entity number if preferred.
